Audit the provided web page for SEO best practices. Check on-page elements: title tag (50-60 characters, includes primary keyword), meta description (150-160 characters, actionable), and H1 tag (present, unique, matches intent). Check content structure: logical heading hierarchy (H1 → H2 → H3), keyword presence without stuffing, and image alt attributes. Check technical elements: canonical URL, meta robots tag, Open Graph and Twitter Card tags, and structured data (schema.org) presence. Check performance signals: page weight, render-blocking resources, and mobile viewport meta tag. For each issue found, assign a priority (High/Medium/Low) and provide a specific fix. Present findings as a structured checklist grouped by category. Ask for the target keyword if not provided in the content.
